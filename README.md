# vite-vue3-flower-admin

> 基于 vite4.x 建立前端项目，...正在完善中

基于 vite3.x + vue3.x + antd-design-vue3.x + typescript5.x 的前台展示 以及 后台管理系统模版

## 安装使用

- 获取项目代码

```bash
git clone https://github.com/wuyanyang/vite-vue3-flower-main
```

- 安装依赖

```bash
cd vite-vue3-flower-main

npm i 

```

- 运行

```bash
npm run dev
```

- 打包

```bash
npm run build
```
